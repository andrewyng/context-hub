# vibrateShort(Object object)

使手机发生较短时间的振动。
**注意事项**：Android 某些机型在不支持短振动，可以使用[vibrateLong](https://open.feishu.cn/document/uYjL24iN/uEDOx4SM4EjLxgTM)代替。

## 支持说明

应用能力 | Android | iOS | PC | Harmony | 预览效果
---|---|---|---|---|---
小程序 | **✓** | **✓** | **X** | V7.35.0+ | 预览
网页应用 | V3.44.0+ | V3.44.0+ | **X** | V7.35.0+ | 预览

## 输入
继承[标准对象输入](https://open.feishu.cn/document/uYjL24iN/ukzNy4SO3IjL5cjM)，无扩展属性

## 输出
继承[标准对象输出](https://open.feishu.cn/document/uYjL24iN/ukzNy4SO3IjL5cjM#8c92acb8)，无扩展属性

## 示例代码

```js
tt.vibrateShort({ 
    success(res) {
      console.log(JSON.stringify(res));
    },
    fail(res) {
      console.log(`vibrateShort fail: ${JSON.stringify(res)}`);
    }
});
```

`success`返回对象示例：
```json
{
    "errMsg": "vibrateShort:ok"
}
```
